TWENTY SEVEN PRO
http://briangardner.com/themes/twenty-seven/

INSTALL
1. Upload the Twenty Seven Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Twenty Seven Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

WIDGET AREAS
Sticky Message - This is the sticky message section.
Before Header - This is the before header section.
Before Footer - This is the before header section.
After Entry - This is the after entry widget area.
